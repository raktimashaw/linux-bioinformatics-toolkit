A collection of Bash scripts developed while learning Linux and Bash for bioinformatics.

This project demonstrates essential Linux command-line skills used in computational biology, including sequence counting, if conditions and loop. 

# Repository Structure

linux-bioinformatics-toolkit/
│
├── scripts/

│   ├── genome_checker.sh
│   └── fasta_summary.sh
│
├── datasets/
│
├── example_output/
│
└── docs/

# Skills Demonstrated

- Linux command line
- Bash scripting
- Variables
- Loops (`for`)
- Conditional statements (`if`)
- File checking (`-f`)
- `grep`
- Command substitution
- Output redirection (`>` and `>>`)
- Report generation
